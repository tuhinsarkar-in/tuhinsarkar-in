# Social Media Icons

This folder contains icon images for the social media links on your portfolio.

## Required Icons:

Place the following icon files in this folder:

1. **youtube.png** - YouTube icon
2. **github.png** - GitHub icon  
3. **linkedin.png** - LinkedIn icon
4. **behance.png** - Behance icon
5. **instagram.png** - Instagram icon
6. **twitter.png** - Twitter/X icon

## Icon Specifications:

- **Format**: PNG (with transparency recommended)
- **Size**: 24x24px or 48x48px (will be resized to 24x24px)
- **Style**: White or light colored icons work best
- **Background**: Transparent background preferred

## Where to Get Icons:

- **Free Icons**: [Flaticon](https://www.flaticon.com), [Icons8](https://icons8.com), [Font Awesome](https://fontawesome.com)
- **Official Brand Icons**: Search for "[brand name] icon png" 
- **Create Your Own**: Use tools like Canva, Figma, or Photoshop

## Current Setup:

The icons will automatically:
- Display at 24x24px size
- Have a slight dim effect (brightness 0.7)
- Brighten and glow on hover
- Scale up slightly on hover

## Note:

Until you add your own icons, the portfolio will show broken image placeholders. The functionality and hover effects are already working - you just need to drop in your icon images!
